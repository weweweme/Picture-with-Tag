# Purpose

- Implement a simple data storage and search module
- The core functionality is a search module based on tags

## Features

### Storage

- Implemented similar to a blog post
- Includes title, picture, tags, and content

### Search

- Searchable by title
- Searchable by tags
- Searchable by text
